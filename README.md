# MSI Packaging Demo using WiX

This project demonstrates how to create an MSI installer from scratch using WiX Toolset.

## What this includes
- Raw application files
- WiX (.wxs) configuration
- MSI build process

## Prerequisites
- WiX Toolset v3.14 installed

## Steps
1. Create demo files
2. Build using candle.exe and light.exe
3. Generate MSI installer

## Output
DemoApp.msi

## Concepts Covered
- EXE vs MSI vs MST
- Basic MSI packaging workflow
